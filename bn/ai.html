AI ও এজেন্ট

আমি LLM-এর সাথে হাতে-কলমে কাজ করি — লোকাল-ফার্স্ট কোডিং এজেন্ট, Claude Code প্লাগইন ও স্কিল, MCP সার্ভার, এবং মাঝে মাঝে নিজের মডেল ভাঙার পরীক্ষা। এই পেজে আমি কী তৈরি করছি, পড়ছি এবং বিশ্লেষণ করছি তা ট্র্যাক করি।

বর্তমান ফোকাস

  • —লোকাল-ফার্স্ট কোডিং এজেন্ট (Ollama, Gemma, ReAct লুপ)
  • —Claude Code প্লাগইন এবং নিজে লেখা স্কিল
  • —বাস্তব ডেভ ওয়ার্কফ্লোর জন্য Model Context Protocol (MCP) সার্ভার
  • —NLP-এ অ্যাডভার্সারিয়াল উদাহরণ এবং মডেল রোবাস্টনেস
  • —ONNX Runtime দিয়ে ব্রাউজার-নেটিভ ইনফারেন্স
